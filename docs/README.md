# serverless-rust-plugin commands

There are some CLI commands for testing and debugging locally. These commands use docker cli internally, so make sure the docker daemon is running on your machine before executing.

- [rust:start](./rust:start)
- [rust:ps](./rust:ps)
- [rust:invoke](./rust:invoke)
- [rust:logs](./rust:logs)
- [rust:stop](./rust:stop)
